Your current .env file only has Supabase config. You need to ADD these variables for Chicago Hub to work locally:

## ADD TO YOUR .env FILE:

# Chicago Hub Configuration
OPENAI_API_KEY=sk-proj-[your-openai-api-key-here]
OPENAI_ASSISTANT_ID=asst_[your-assistant-id]
VECTOR_STORE_ID=vs_[your-vector-store-id]

VITE_LAMBDA_AI_ASSISTANT_URL=http://localhost:3002

# AWS S3 Configuration
AWS_ACCESS_KEY_ID=AKIA[your-aws-access-key]
AWS_SECRET_ACCESS_KEY=[your-aws-secret-access-key]
AWS_REGION=us-east-2
AWS_S3_BUCKET=chicago-hub

FRONTEND_URL=http://localhost:8080

# JWT Configuration
JWT_SECRET=soawesomevibecoding

# MongoDB Configuration (THIS IS CRITICAL!)
MONGODB_URI=mongodb+srv://[username]:[password]@cluster0.1shq5cl.mongodb.net/?retryWrites=true&w=majority&appName=Cluster0

# Mailgun Email Configuration
MAILGUN_DOMAIN=[your-mailgun-domain]
MAILGUN_API_KEY=[your-mailgun-api-key]
MAILGUN_BASE_URL=https://api.mailgun.net/v3
MAILGUN_FROM_EMAIL=noreply@empowerlocal.co
MAILGUN_FROM_NAME=Chicago Hub
ADMIN_EMAIL=admin@empowerlocal.com

# Server Configuration
PORT=3001
NODE_ENV=development

## STEPS TO RESTORE:

1. Open your .env file in your editor
2. Keep the existing Supabase variables at the top
3. Add all the variables above to the bottom
4. Save the file

## THEN RUN LOCALLY:

# Start the backend server
npm run server:dev

# In another terminal, start the frontend
npm run dev

Your app will be available at:
- Frontend: http://localhost:8080
- Backend: http://localhost:3001
