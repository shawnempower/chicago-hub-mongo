## Behavior

### On Category Selection
1. Radio button becomes selected
2. "Add Category" button becomes enabled
3. Click "Add Category" to proceed

### After Adding
1. Modal closes
2. New tab appears in correct position
3. Tab opens automatically in EDIT mode
4. Empty state shows with guidance
5. Shows relevant "Add [Sub-section]" buttons

### Empty States by Category

**Digital**:
```
Get started by adding your website or email newsletter

[+ Add Website Advertising]  [+ Add Email Newsletter]
```

**Print**:


**Streaming**:
```

```

**Broadcast**:

## Validation

- Only allow adding each category once
- If category already exists, show message: "This category already exists"
- Gray out already-added categories in modal
